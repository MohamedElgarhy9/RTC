# RTC
## About the project:
    Real Time Clock, it is an implementation of real time clock using Atmega32A mc, LCD, Keypad and 7 segments
### Major components:
  #### External:
    - ATmega32A microcontroller 
    - 2*16 LCD
    - Keypad
    - 7 segment
  #### Internal:
    - DIO
    - Timer1
